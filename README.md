
# Change Extensions

An application that helps us to change the extension of one or several files at the same time.
## Run Locally

Clone the project

```bash
  git clone git@github.com:reliek21/change_extensions.git
```

Go to the project directory

```bash
  cd change_extensions
```

Start

```bash
  python3 script.py
```
